# AI Medical Diagnosis Assistant

Tıbbi görüntüleme verileri ve hasta semptomlarını analiz ederek doktorlara karar destek mekanizması sunan yapay zeka asistanıdır.

## Modüller
1. **Görüntü Analizi:** Radyolojik görüntülerin CNN modelleri ile incelenmesi.
2. **Tanı Tahmini:** Semptom tabanlı olasılıksal tanı modelleri.
3. **Otomatik Raporlama:** Bulguların teknik dokümantasyon standartlarına uygun şekilde özetlenmesi.

## Teknik Altyapı
- **Derin Öğrenme:** TensorFlow ve Keras tabanlı özel mimariler.
- **Veri Analizi:** Pandas ile tıbbi veri setlerinin işlenmesi.
- **Sınıflandırma:** VGG16/ResNet tabanlı transfer öğrenme yaklaşımları.
